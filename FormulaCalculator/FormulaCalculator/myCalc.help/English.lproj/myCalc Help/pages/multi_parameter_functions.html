Multi parameter functions

Multi parameter functions are functions that obviously takes more than one parameter. Such functions are available in the side drawers to calculate more complex functions, like the monthly payments of an annuity loan, that takes four parameters. 

When you select those functions, you enter into a â€œparameter collection mode", where the parameter asked for is explained in the yellow message field of the calculator. When the parameter is entered as a number or an equation, press NEXT to go on with the next parameter. The bottom drawer tells about all the parameters needed.  

Because you can have formulas as parameters to such functions, one can have no other multi-parameter functions as parameter. This is a limitation of the user interface and not in the calculation engine. In rare cases you need it, you can put it in like two formulas, you can edit afterwards where you copy and paste one formula into as parameter in the other. Other functions and operators can be used right away. 

The multi parameter functions are more closesly described under the chapters of financial, statistical or gemetry functions.
