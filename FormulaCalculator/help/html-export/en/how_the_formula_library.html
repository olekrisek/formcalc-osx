How the formula library

There is many library functions you can use to calculate common tasks. Most of those functions are ordinary formulas stored in the program. When you use those functions, the name of the function is put into your formula area with an opening parenthesis. They typically take one to five parameters separated with semi-colon: ";". Every parameter can either be a single numeric value, or a formula. 

When you select those functions, you enter into a â€œparameter collection mode", where the parameter asked for is explained in the yellow message field of the calculator. When the parameter is entered as a number or an equation, press TAB to go on with the next parameter. When all parameters are collected, the function is closed with the closing parenthesis. You are not suposed to close the function with the closing parenthesis yourself, as completing the last parameter and the TAB key, will do it for you. 

The calculation engine are shut of while collecting parameters, since this obviously makes no sence to present partially result before all the parameters are collected. What you see in the display is therefore only the numbers you enter. When the last parameter is collected, the calculation engine starts again, calculating the complete formula. 

The bottom drawer tells about all the parameters needed.  The text in the bottom drawer can be selected and copied for documentation purposes. 

When you press the "=" - the function is documented on the print-roll, explaining the parameters, the formulas used to calculate, and the result of the calculation. The reason for this documentation is that it might be needed for some document you are working on.

You can, enter such function in EDIT mode manually. The function name and the number of parameters must be correct. The parameters are separated by a ";". Commonly used functions can be cut and paste from stickies, or other means of storing commonly used text. This applies to complete formulas you oftenly use as well.
