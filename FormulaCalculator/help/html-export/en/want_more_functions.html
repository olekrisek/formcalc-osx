I have provided a basic set of functions in this calculator app, but this can easily be extended. Drop me a line at olekrisek@gmail.com, and tell what standard formulas or functions you would like to have.

Actual functions to implement, is functions that basically return one result, with one or more parameters as input. The calculation method can either be:

  • a direct calculation,
  • an iteration style calculation where a guessed answer is input in a complex formula over and over again until we find the correct value. Such methods are used where the formula cant be re-ordered so that we got the desired value alone on one side of the equation.
  • A simulation style calculation, where many calculations is done in series until we find the result.
