Multi parameter functions

Multi parameter functions are functions that obviously takes more than one parameter. Such functions are available in the side drawers to calculate more complex functions, like the monthly payments of an annuity loan, that takes four parameters. When you select those functions, you enter into a â€œparameter collection mode", where the parameter asked for is explained in the yellow message field of the calculator. When the parameter is entered as a number or an equation, press NEXT to go on with the next parameter. The bottom drawer tells about all the parameters needed.
